# GUIアプリ（Flow Cut フロントエンド）実装プラン

**作成日:** 2025-11-25
**バージョン:** 1.0
**ステータス:** 進行中（Phase 1 完了）

既存の CLI ベースの文字起こしパイプラインを GUI から操作できるようにするための段階的プラン。Tkinter を用いたデスクトップアプリとして実装し、将来の Web 版へ発展させやすい構造を維持する。

---

## フェーズ進捗
- [x] **Phase 1: 最小GUI（ファイル選択＋実行ボタン）**
  - Tkinter ベースのメインウィンドウを追加
  - CLI パイプライン (`execute_poc_run`) をコントローラ経由で呼び出し
  - ステータス表示・簡易プログレスバーを実装
- [ ] Phase 2: ステータス詳細化（工程別の進捗表示）
- [ ] Phase 3: オプション設定UI（LLM選択、リライト切替 等）
- [ ] Phase 4: Web版検討・実装

---

## 目的と要件要約
- CLI ロジックを流用し、GUI側でビジネスロジックを持たない。
- Tkinter + ttk を用い、最小限の UI （ファイル選択、実行ボタン、簡易ステータス）を提供。
- 実行は別スレッドで行い、UI フリーズを防ぐ。

---

## 操作手順（Phase 1）
1. 依存をインストールし、必要に応じて `.env` を準備。
2. 以下のコマンドで GUI を起動する:
   ```bash
   python -m src.cli.main gui
   ```
3. ウィンドウで音声ファイルを選択し、「実行」を押下。
4. ステータスが「待機中 → 処理中… → 完了しました」と遷移し、出力ファイルパスが表示される。
