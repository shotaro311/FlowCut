# LLM 設定ファイル群

FlowCut の LLM 周りの設定は、コードから分離してこのディレクトリで管理します。

- `llm_profiles.json`  
  - LLM プロバイダーとパスごとのモデル構成をまとめた「プロファイル」定義です。  
  - 例: `default` プロファイルでは Google + Gemini の組み合わせ、`openai_gpt5` では OpenAI + GPT-5 系モデルなど。
  - 将来的に GUI からプロファイルを選択・編集しやすい構造にしておきます。

- `llm_pricing.json`  
  - 各プロバイダー/モデルごとの **1Mトークンあたりの概算料金** を保持します。  
  - メトリクス出力（`logs/metrics/*.json`）で、おおよその推定コストを計算するために使用します。
  - 実際の金額は各社の料金ページに従い、必要に応じて手動で更新してください。
