# src ディレクトリ

Flow Cut コア機能のPython実装を配置します。モジュールは用途別に細分化し、CLI層・設定層・文字起こし処理・LLM整形・アライメントなどをここからインポートします。

- `cli/`: TyperベースのCLI実装（後日追加）
- `config/`: `.env` 読み込みと共有設定
- `transcribe/`: 音声認識ランナーとPoC
- `llm/`: LLM呼び出しとプロンプト生成
- `alignment/`: タイムスタンプ調整
- `utils/`: ログ/進捗管理

各サブディレクトリにも README を置き、責務を明記してください。
