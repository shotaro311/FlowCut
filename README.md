# FlowCut（フローカット）

FlowCut は、動画や音声ファイルから「字幕」を自動で作ってくれる便利なソフトです。
難しい設定は不要で、Windows パソコンで簡単に使うことができます。

## 🌟 このソフトでできること

*   **動画から文字起こし**: お手持ちの動画や音声ファイルから、自動でしゃべっている内容を書き出します。
*   **きれいな字幕に整える**: AI が「えー」「あのー」といった言葉を取り除き、読みやすい文章に整えてくれます。
*   **字幕ファイル（.srt）の作成**: YouTube や動画編集ソフトでそのまま使える字幕ファイルができあがります。

---

## 🚀 使い方（かんたん 5 ステップ）

1.  **アプリを開く**: `FlowCut.exe` をダブルクリックして起動します。
2.  **APIキーの設定**: 右上の「API設定」ボタンを押し、お手持ちの API キーを貼り付けて「保存」します（初回のみ）。
3.  **ファイルを選ぶ**: 「ファイルを選択」ボタンを押して、文字起こししたい動画や音声を選びます。
4.  **実行**: 青色の「▶ 実行」ボタンを押します。あとは終わるのを待つだけです。
5.  **完了**: 終わると、選んだ動画と同じ場所に字幕ファイルが作られます。

## 🛠 困ったときは（トラブルシューティング）

### Windows で起動時にエラーが出る場合
「**VCRUNTIME140.dll が見つからない**」や「**MSVCP140.dll がない**」などのエラーが出て起動できないときは、以下のマイクロソフト公式サイトから「x64」版をダウンロードしてインストールしてください。

👉 [Microsoft 公式ダウンロード（vc_redist.x64.exe）](https://aka.ms/vc14/vc_redist.x64.exe)

※インストール後、もう一度アプリを起動してみてください。

より詳しい解決方法は、`docs/README.md` の「Windows: 起動できない / dll が見つからない」に記載されています。
