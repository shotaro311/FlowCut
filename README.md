# FlowCut

FlowCut は、音声/動画ファイルから日本語字幕（`.srt`）を作るデスクトップアプリです。  
ローカルで Whisper による文字起こしを行い、必要に応じて LLM（Google/OpenAI/Anthropic）で字幕用に整形します。

## できること

- 音声/動画ファイルを読み込み（動画は必要に応じて音声を抽出）
- ローカルで文字起こし（Whisper）
- 字幕用に整形（改行、誤字・フィラー除去、Glossary 反映など）
- `.srt` 出力、処理時間/トークン/概算コスト表示、ログ保存（任意）

## 使い方（アプリ利用者）

1. アプリを起動
2. 右上の「API設定」で、利用するプロバイダーの API キーを保存
3. 「ファイルを選択」→（必要なら）「保存先を変更」→「▶ 実行」
4. 完了後、保存先フォルダに `.srt` が生成されます

詳しい手順は `docs/README.md` を参照してください。

## 注意点

- LLM 整形を使う場合はインターネット接続が必要です（料金は各プロバイダーの課金）
- 初回はモデル準備やキャッシュ作成で時間がかかることがあります
- 動画入力は内部で音声抽出します（配布物の構成によっては ffmpeg が必要になる場合があります）

## 開発者向け（このリポジトリから実行する場合）

- GUI 起動: `python -m src.cli.main gui`
- CLI 実行: `python -m src.cli.main run <audio_or_video>`
- 詳細は `docs/runbook.md` を参照してください
